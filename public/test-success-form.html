✅ SSLCommerz Success Page Test (POST Method)

This test properly simulates the SSLCommerz success callback using POST method to verify the "Your Payment is Successful!" message is displayed correctly.

📋 Test Information

Expected Message: "Your Payment is Successful!"

Test Method: POST request (as required by SSLCommerz)

Route: /success (POST only)

Note: The success route only accepts POST requests for security reasons.

🔧 Test Parameters

🎯 How to Use This Test

  1. Fill in the test parameters (or use the defaults)
  2. Click the "Test Success Page (POST)" button
  3. The form will submit a POST request to /success
  4. You will be redirected to the success page
  5. Verify that "Your Payment is Successful!" message is displayed
